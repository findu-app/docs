# Partner Dashboard API

The Partner Dashboard API enables schools, counselors, and partner organizations to manage their presence on FindU and engage with prospective students.

## Authentication

All endpoints require JWT authentication with appropriate partner permissions:

```bash
Authorization: Bearer YOUR_JWT_TOKEN
X-Entity-ID: YOUR_ENTITY_ID
```

## Analytics Endpoints

### Get Dashboard Metrics

Retrieve analytics data for your institution.

<CodeGroup>
```bash cURL
curl -X GET "https://api.findu.app/v2/partner/analytics?period=30d" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "X-Entity-ID: YOUR_ENTITY_ID"
```

```javascript JavaScript
const analytics = await fetch('https://api.findu.app/v2/partner/analytics?period=30d', {
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'X-Entity-ID': 'YOUR_ENTITY_ID'
  }
});

const data = await analytics.json();
```
</CodeGroup>

#### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| period | string | Time period: "7d", "30d", "3m", "1y" |
| metric | string | Specific metric to retrieve |

### Export Campaign Data

Export detailed student interaction data.

```
POST /v2/partner/campaigns/{campaign_id}/export
```

#### Request Body

```json
{
  "format": "csv",
  "fields": ["name", "gpa", "test_scores", "interaction_date", "match_score"],
  "filters": {
    "interaction_type": ["view", "like"],
    "date_range": {
      "start": "2024-01-01",
      "end": "2024-01-31"
    }
  }
}
```

## Student Management

### List Campaign Students

Get students who have interacted with your institution.

```
GET /v2/partner/campaigns/{campaign_id}/students
```

#### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| page | integer | Page number (default: 1) |
| limit | integer | Results per page (max: 100) |
| sort | string | Sort by: "recent", "match_score", "gpa" |
| filter | object | Filter criteria |

### Upload Media

Upload images or videos for your profile.

```
POST /v2/partner/media
```

#### Request

Multipart form data with:
- `file`: The media file
- `type`: "image" or "video"
- `caption`: Optional description

#### Response

```json
{
  "media": {
    "id": "media_123",
    "url": "https://storage.findu.app/...",
    "thumbnail_url": "https://storage.findu.app/...",
```

## Team Management

### Invite Team Member

Add a new user to your partner account.

```
POST /v2/partner/team/invite
```

#### Request Body

```json
{
  "email": "colleague@example.edu",
  "role": "counselor",
  "entities": ["entity_123", "entity_456"],
  "permissions": {
    "can_message": true,
    "can_view_analytics": true,
    "can_manage_profile": false
  }
}
```

### Update Member Permissions

Modify team member access.

```
PATCH /v2/partner/team/members/{member_id}
```

#### Request Body

```json
{
  "role": "admin",
  "entities": ["entity_123"],
  "permissions": {
    "can_message": true,
    "can_view_analytics": true,
    "can_manage_profile": true,
    "can_invite_members": true
  }
}
```

## Webhooks

### Register Webhook

Set up webhooks for real-time events.

```
POST /v2/partner/webhooks
```

#### Request Body

```json
{
  "url": "https://example.edu/webhooks/findu",
  "events": ["student.liked", "message.received", "student.viewed"],
  "secret": "webhook_secret_key"
}
```

### Webhook Events

| Event | Description |
|-------|-------------|
| student.viewed | Student viewed your profile |
| student.liked | Student liked your school |
| message.received | New message from student |
| scholarship.saved | Student saved scholarship |

#### Event Payload

```json
{
  "event": "student.liked",
  "timestamp": "2024-01-15T10:30:00Z",
  "data": {
    "student_id": "123e4567-e89b-12d3-a456-426614174000",
    "entity_id": "entity_123",
    "metadata": {
      "match_score": 0.87,
      "source": "swiping"
    }
  }
}
```

## Rate Limits

| Endpoint Type | Limit |
|---------------|-------|
| Analytics | 100/minute |
| Messaging | 60/minute |
| Profile Updates | 20/minute |
| Exports | 5/minute |

## Error Handling

```json
{
  "error": {
    "code": "RATE_LIMITED",
    "message": "Too many requests",
    "retry_after": 30
  }
}
```

## SDKs

### JavaScript/TypeScript

```typescript
import { FindUPartnerClient } from '@findu/partner-sdk';

const client = new FindUPartnerClient({
  apiKey: 'YOUR_API_KEY',
  entityId: 'YOUR_ENTITY_ID'
});

// Get analytics
const analytics = await client.analytics.getDashboard({ period: '30d' });

// Send message
await client.messaging.send({
  recipientIds: ['student_123'],
  message: {
    subject: 'Welcome!',
    body: 'Thanks for your interest...'
  }
});
```
