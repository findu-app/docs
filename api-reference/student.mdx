# Student App API

The Student App API powers the FindU iOS application, providing endpoints for authentication, profile management, college discovery, and messaging.

## Authentication

### Sign In with Apple

Authenticate using Apple Sign In.

<CodeGroup>
```swift Swift
let appleIDProvider = ASAuthorizationAppleIDProvider()
let request = appleIDProvider.createRequest()
request.requestedScopes = [.fullName, .email]

// Handle authorization
let authController = ASAuthorizationController(authorizationRequests: [request])
authController.delegate = self
authController.performRequests()
```

```bash API Call
curl -X POST https://api.findu.app/v2/auth/apple \
  -H "Content-Type: application/json" \
  -d '{
    "id_token": "APPLE_ID_TOKEN",
    "nonce": "NONCE_VALUE"
  }'
```
</CodeGroup>

#### Response

```json
{
  "session": {
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "refresh_token": "refresh_token_value",
    "expires_in": 3600,
    "user": {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "email": "student@example.com",
      "has_profile": false
    }
  }
}
```

### Sign In with Google

Authenticate using Google Sign In.

```
POST /v2/auth/google
```

#### Request Body

```json
{
  "id_token": "GOOGLE_ID_TOKEN",
  "access_token": "GOOGLE_ACCESS_TOKEN"
}
```

### Email/Password Auth

Traditional email authentication.

```
POST /v2/auth/email
```

#### Request Body

```json
{
  "email": "student@example.com",
  "password": "secure_password"
}
```

### Refresh Token

Refresh an expired access token.

```
POST /v2/auth/refresh
```

#### Request Body

```json
{
  "refresh_token": "REFRESH_TOKEN"
}
```

## Error Responses

### Standard Error Format

```json
{
  "error": {
    "code": "PROFILE_INCOMPLETE",
    "message": "Please complete your profile to access this feature",
    "details": {
      "missing_fields": ["gpa", "test_scores"]
    }
  }
}
```

### Common Error Codes

| Code | Description |
|------|-------------|
| UNAUTHORIZED | Invalid or expired token |
| PROFILE_INCOMPLETE | Missing required profile data |
| SCHOOL_NOT_FOUND | Invalid school ID |
| RATE_LIMITED | Too many requests |
| VALIDATION_ERROR | Invalid request data |

## iOS SDK Integration

### Swift Example

```swift
import FindUSDK

class CollegeService {
    let client = FindUClient.shared
    
    func loadRecommendations() async throws -> [School] {
        let response = try await client.recommendations.fetch(
            limit: 20,
            excludeSeen: true
        )
        return response.schools
    }
    
    func recordSwipe(schoolId: String, liked: Bool) async throws {
        try await client.interactions.swipe(
            schoolId: schoolId,
            action: liked ? .like : .dislike,
            timeSpent: 10.5
        )
    }
}
```
