# Quick Start Guide

Get your FindU development environment set up and make your first contribution.

## Prerequisites

Before you begin, ensure you have:

<CardGroup cols={2}>
  <Card title="Required Tools" icon="wrench">
    - Git (v2.30+)
    - Node.js (v18+ LTS)
    - Supabase CLI
    - Xcode (for iOS development)
  </Card>
  <Card title="Access Needed" icon="key">
    - GitHub organization access
    - Supabase credentials
    - Slack workspace invite
    - Railway access (optional)
  </Card>
</CardGroup>

<Note>
  Don't have access yet? See our [onboarding checklist](/developer/onboarding) for new team members.
</Note>

## Automated Setup

We've created a script that sets up your entire development environment:

```bash
# Clone the dev-tools repository
git clone https://github.com/findu-app/dev-tools.git

# Run the setup script
cd dev-tools
./scripts/setup.sh
```

This script will:
1. Clone all FindU repositories
2. Install dependencies for each project
3. Create your `.env.local` configuration
4. Set up pre-commit hooks
5. Configure the findu CLI tool

## Manual Setup (Alternative)

If you prefer manual setup or need to troubleshoot:

<Steps>
  <Step title="Clone Repositories">
    ```bash
    mkdir ~/findu && cd ~/findu
    git clone https://github.com/findu-app/ios_app.git
    git clone https://github.com/findu-app/web_app.git
    git clone https://github.com/findu-app/supabase.git
    git clone https://github.com/findu-app/matching-algorithm.git
    ```
  </Step>
  
  <Step title="Install Dependencies">
    ```bash
    # Web app
    cd web_app && npm install
    
    # iOS app
    cd ../ios_app && pod install
    
    # ML engine
    cd ../matching-algorithm
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```
  </Step>
  
  <Step title="Configure Environment">
    Create `~/.env.local` with your credentials (get from team lead)
  </Step>
</Steps>

## Verify Your Setup

Run these commands to verify everything is working:

```bash
# Check environment
./findu env status

# Run web app
cd ~/findu/web_app && npm run dev

# Check database connection
cd ~/findu/supabase && supabase status
```

## Make Your First Change

<Steps>
  <Step title="Switch to dev branch">
    ```bash
    cd ~/findu/web_app
    git checkout dev
    git pull origin dev
    ```
  </Step>
  
  <Step title="Create feature branch">
    ```bash
    git checkout -b feature/your-name-first-pr
    ```
  </Step>
  
  <Step title="Make a small change">
    Edit any file (try updating a comment or fixing a typo)
  </Step>
  
  <Step title="Commit and push">
    ```bash
    git add .
    git commit -m "My first FindU commit"
    git push -u origin feature/your-name-first-pr
    ```
  </Step>
  
  <Step title="Open PR">
    Go to GitHub and create a pull request targeting the `dev` branch
  </Step>
</Steps>
